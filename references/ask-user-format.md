# AskUserQuestion Format

All user-facing questions must follow this structure:

```
**Re-ground:** One sentence explaining where we are in the workflow.

**Context:** Brief explanation of why this question matters.

**Options:**
1. **Option A** — what it does (recommended if applicable)
2. **Option B** — what it does
3. **Option C** — what it does (if applicable)

**Recommendation:** Which option and why (one sentence).
```

## Rules

- Always provide a recommendation — don't make the user think when you have enough context to suggest.
- Keep options to 2-4. More than 4 means you need to pre-filter.
- Use "recommended" tag on the option you'd pick.
- If there's truly only one sensible path, state it and ask for confirmation rather than presenting fake options.
